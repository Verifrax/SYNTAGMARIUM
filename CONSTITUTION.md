# SYNTAGMARIUM Constitution

## System axiom

No active Verifrax surface may contradict the reconciled epoch.

## Constitutional split

- SYNTAGMARIUM defines law.
- ORBISTIUM stores accepted state.
- CONSONORIUM evaluates observed reality against law.

## Absolute invariants

1. One active truth per governed object.
2. Historical surfaces must self-identify.
3. Unknown state is never silently healthy.
4. One host has one owner and one primary role.
5. One package maps to one source repository.
6. One repository has one primary role.
7. Projection surfaces are subordinate to law and epoch.
8. Apache-2.0 is the default governed license.
9. The durable default branch is main unless law explicitly grants an exception.
10. Publicly relevant entry surfaces must be legible in under 30 seconds.
11. No governed object class is admitted without an explicit claim-class definition.
