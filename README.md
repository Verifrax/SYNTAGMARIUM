# SYNTAGMARIUM

SYNTAGMARIUM is the constitutional law repository for the Verifrax system.

## What this repository is

SYNTAGMARIUM defines the lawful grammar of the Verifrax system:
- object schemas
- role catalog
- invariants
- severity rules
- transition rules
- projection contracts
- exception rules

## What this repository is not

SYNTAGMARIUM does not store live world state.
SYNTAGMARIUM does not execute reconciliation.
SYNTAGMARIUM does not publish epochs.
SYNTAGMARIUM does not collect GitHub, npm, HTTP, or evidence telemetry.

## Dependency direction

SYNTAGMARIUM -> ORBISTIUM
SYNTAGMARIUM -> CONSONORIUM

Law originates here.
State does not define law.
Runtime does not define law.

No active Verifrax surface may contradict the reconciled epoch.
